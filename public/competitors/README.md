## How to add

1. Visit each competitor's AI product page.
2. Capture a screenshot (hero section or main value proposition).
3. Save as PNG or JPG with the filename above.
4. Place in this folder.

If no screenshot is present, the presentation will show a visual mock based on their positioning data.
